# Backend API

This is a simple Express backend with a Hello World API.

## Usage

1. Install dependencies:
   ```sh
   npm install
   ```
2. Start the server:
   ```sh
   npm start
   ```
3. Access the API at [http://localhost:3000/api/hello](http://localhost:3000/api/hello)



To run your backend:

Open a terminal and navigate to the backend folder:
cd backend

Install dependencies:
npm install

Start the server:
npm start

Your API will be available at http://localhost:3000/api/hello.
